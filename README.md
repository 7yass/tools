# tools.brazy.xyz

roadmaps, mastermaps, and every project in the brazy ecosystem.

## sections

- `/roadmaps` — step-by-step mastermaps for every stack and skill
- `/projects` — every brazy build, linked and documented

## stack

- next.js 15 (app router)
- typescript
- vercel (deploy)

## adding content

**new roadmap** — edit `lib/roadmaps.ts`, add an object to the `roadmaps` array

**new project** — edit `lib/projects.ts`, add an object to the `projects` array

routes auto-generate from the slugs.
